## 🎮 About

**FileLink: WiFi File Transfer** transforms your mobile device into a local server, allowing you to access, upload, and download files wirelessly via any web browser. No cables, no mobile data usage—just seamless local network transfer.

🚀 Official release by **Lihka Games**!

---

## ⚡ Key Features

| Feature | Description |
|---------|-------------|
| 🔗 **Web-Based UI** | Access files directly from your desktop browser |
| 🛡️ **PIN Protection** | Secure your session with a 4-digit PIN |
| ⚡ **QR Quick Connect** | Instantly pair devices by scanning the QR code |
| 📂 **Categorized** | Automatically sorts Images, Videos, and Docs |

| Component | Detail |
|---------|--------|

| ⏳ **Timeout** | Automatic session expiration for privacy |
